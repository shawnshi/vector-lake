# Value Translation (价值翻译)

## 定义
价值翻译是指将 AI 系统的技术性能指标（如准确率、召回率、推理延迟）转化为医院管理决策层能够理解并愿意付费的**经营结果指标**（如创收能力、成本降减、安全保全、品牌价值）的过程。

## 院长驾驶舱报告模型 (Director Cockpit Model)
针对医疗机构核心决策层（院长），价值量化应遵循“五大模块”结构以实现高效翻译[^1]：
1. **燃点标题**：直击核心痛点（如“年均节省书记员 5000 工时”）。
2. **黄金摘要**：30秒内清晰陈述投入产出比。
3. **价值仪表盘**：可视化对比应用前后的经营指标（工时、纠纷率、DRG 盈余）。
4. **用户证言**：临床一线医生的真实体感反馈。
5. **下一步建议**：驱动持续投入的行动指南。

## 翻译矩阵
| 技术指标 (过程世界) | 管理指标 (结果世界) |
| :--- | :--- |
| 病历生成准确率 95% | 每位医生每天节省 1.5 小时文书时间 -> 可增加 10% 门诊放号量 |
| RAG 召回率提高 | 医疗纠纷风险降低导致的赔付预估减少 |
| DRG 分组预测准确 | 医保违规核减金额降低 15% |

## 核心方法论
*   **双语者机制**：需要具备跨界背景的咨询顾问进行中介。
*   **逆向推演**：从院长的 KPI 出发，倒推技术点能如何影响对应指标。
*   **定位时间黑洞**：通过实地调研挖掘隐形效率损耗点 [支撑:: [[Concept_Time_Black_Hole_Mining.md]]]。
