# OmniHuman

## 概览
OmniHuman 是一个通用的音频驱动人物动画生成模型。它突破了传统数字人生成对极高质量、干净训练数据的依赖。

## 关键技术
- **强弱条件混合训练**: 通过引入文本、姿态等强约束条件，带动音频等弱约束条件的学习，实现了对复杂场景视频数据的有效利用 [^1]。

## 医疗场景应用
- **虚拟护士交互**: 提供更加自然、富有情感反馈的医患交互界面。
- **康复指导可视化**: 将枯燥的文字医嘱自动转化为高度拟人化的视频指导，提升患者依从性。
