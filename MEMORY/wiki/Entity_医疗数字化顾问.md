# 医疗数字化顾问 (Medical Digitalization Consultant)

## 定义与定位
医疗数字化顾问是被重新定义的专业主体，其核心使命是从传统的“系统销售”转型为“问题诊断医生”。其价值不仅在于交付软件，更在于通过 **[[Concept_结构化思维_咨询手术刀]]** 对医院运营中的混沌痛点进行“手术级”解剖，并提出具备业务增长逻辑的数字化蓝图[^1]。

## 核心能力栈
- **问题诊断力**：利用 **[[Concept_MECE_原则]]** 将情绪化抱怨转化为量化指标。
- **架构规划力**：通过 **[[Concept_四大架构]]** 设计实现愿景的骨架。
- **价值呈现力**：利用 **[[Concept_价值故事线]]** (SCARI模型) 争取管理层支持。

## 关键价值观
- **诊断优于方案**：看清病因是开方的前提。
- **愿景驱动实施**：技术服务于业务目标的达成。
