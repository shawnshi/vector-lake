# 第二讲：能力真相 —— 作为“智力杠杆”的LLM

## 核心观点
本讲重新定义了 LLM 在医疗场景中的核心价值：它不是单纯的功能集合，而是通过规模化处理非结构化信息，扮演**智力杠杆 (Intelligence Lever)** 的角色[^1]。

## 关键内容
- **能力升维与初稿革命**：LLM 的本质在于革命性降低文书的**[[Concept_Cost_of_First_Draft.md]]**。AI 的核心价值在于将医生从“创造者”降维为“审核者”，消除面对空白文档的认知摩擦[^1]。
- **资源重分配**：通过降低信息同步和文书生成成本，实现医院智力资源的重新配置。文书工作的工业化使得高价值人力能够回流至复杂决策场景[^1]。
- **智力效率**：AI 作为一种杠杆工具，其价值不在于替代专家，而在于撬动现有组织架构中的冗余环节，提升整体系统的单位产出[^1]。
