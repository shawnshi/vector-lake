# Art, Emmie, Penny (Epic AI Assistants)

Art, Emmie 和 Penny 是 [[Entity_Epic_Systems.md]] 推出的“三位一体”AI 助手家族，代表了医疗软件从“工具属性”向“智能实体”的跨越。

## 角色定义与分工

| 助手名称 | 专注领域 | 核心职责 |
| :--- | :--- | :--- |
| **Art** | 临床医学 (Clinical) | 辅助诊断建议、生成电子病历、预测病情恶化风险。 |
| **Emmie** | 患者交互 (Patient Engagement) | 自动回复患者咨询、挂号引导、院后健康随访。 |
| **Penny** | 财务费控 (Financial/RCM) | 医保拒付自动申诉、费用审计、收入循环管理优化。 |

## 战略意义

*   **虚拟员工 (Virtual Workforce)**: Epic 明确将这些助手定义为拥有特定角色和责任边界的“虚拟员工”，而非简单的功能插件。这要求医院从“软件管理”转向“数字劳动力管理”。
*   **ACI 环境感知**: 这些助手利用 [[Concept_ACI_环境感知临床智能.md]] 深度嵌入 Epic 的工作流，实现了“无感集成”[^1]。
*   **商业模式重塑**: 通过这些助手，Epic 将其商业模式从传统的许可证费延伸到了基于智能体能力的订阅费（Agentic Economy）。
