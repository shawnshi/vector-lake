# Pear-001 (Somryst)

## 概述
Pear-001（商品名 Somryst）是全球首款获得 FDA 批准的用于治疗慢性失眠的处方数字疗法（PDT）。它代表了 **[[Concept_数字疗法_DTx]]** 领域从“健康管理软件”向“软件即药物”跨越的里程碑[^1]。

## 核心机制
- **技术基础**：基于失眠认知行为疗法（CBT-I）。
- **交付方式**：通过移动应用引导患者进行睡眠限制、刺激控制和认知重组。
- **临床证据**：通过大规模 RCT 验证，其疗效与传统面授 CBT-I 相当。

## 行业意义
- **监管路径**：确立了软件作为“医疗器械”进行处方级审批的范式。
- **支付尝试**：推动了数字疗法进入医保或商保支付目录的早期探索。
