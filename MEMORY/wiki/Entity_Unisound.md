# 云知声 (Unisound)

## 核心定位
云知声是一家专注于智能语音与 AI 商业化落地的硬核技术公司。在医疗领域，其以“专科定制化”作为核心竞争策略[^1]。

## 核心优势
- **专科深潜**：针对口腔科、手术室等强双手占用场景，提供高精度的专科语音识别与病历生成服务[^1]。
- **私有化算力**：强调在边缘端执行模型推理，保障医院数据主权与响应实时性。

## 市场态势
- 与 [[Entity_iFLYTEK.md]] 在高端三甲医院的专科场景中展开激烈的差异化竞争[^1]。
