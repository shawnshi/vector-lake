# 第十讲：场景决策 —— 从评估矩阵到“一页纸立项书”

## 核心摘要
本讲提供了一套量化的场景筛选工具。核心目标是帮助项目负责人从数百个 AI 想法中识别出最具确定性的“金矿区”，并以极致精简的方式赢得院方的投资承诺[^1]。

## 关键提取
- **四象限决策矩阵**：
    - 横轴：业务价值（临床增益、合规降噪、利润创造）。
    - 纵轴：实施难度（数据可及性、算法复杂度、政治阻力）。
    - **金矿区 (Gold Mine)**：高价值/低难度，首选试点[^1]。
- **一页纸立项书 (One-Page Proposal)**：
    - 核心结构：现状痛点 -> AI 方案 -> 预期收益 -> 所需资源。
    - 设计初衷：让院长在 60 秒内看清投入产出比[^1]。
