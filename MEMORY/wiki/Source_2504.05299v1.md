# Source: Agentic Tree Search: AI Scientist v2

由 Sakana AI 推动的 AI Scientist v2 展示了 AI 从“生成摘要”向“自主执行闭环科研”的跨越。

## 核心架构：Agentic Tree Search
AI Scientist v2 的核心逻辑是将科研实验抽象为树状搜索。
- **并行实验分支**：AI 可以同时启动多个实验方向。
- **自动 Debug 节点**：当代码运行失败时，AI 会创建一个 Debug 节点，尝试修复并重新路径搜索。
- **视觉质检 (VLM-based Audit)**：使用 VLM 对生成的实验图表进行分析，判断实验结果是否合理，而非仅仅依赖文本指标。

## 标志性成果
- **第一篇通过同行评审的 AI 论文**：在 ICLR 2025 ICBINB Workshop 中，AI Scientist v2 生成的论文成功被录用。虽然目前成果处于 Workshop 级别（被称为“高质量灌水”），但其全流程自主性（构思、实验、撰写、修稿）已得到验证。

## 行业启示
这种“暴力试错+逻辑优选”的模式预示着“科研平庸化”时代的到来。对于医疗行业，这意味着临床数据的二次挖掘将进入自动化工厂模式。产生的实验轨迹和 Debug 记录是理想的固态逻辑资产。 [关联:: [[Logic Lake (逻辑湖)]]]
