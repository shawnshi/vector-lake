# 第二十四讲：数字疗法（DTx）与 RWE：超越医院围墙的新范式

## 核心摘要
本讲探讨了医疗数字化的前沿领域：**数字疗法 (DTx)** 与 **真实世界证据 (RWE)**。文章指出，数字化正在打破医院围墙，通过“软件即药物”和“真实诊疗数据”重塑治疗与监管范式[^1]。

## 关键提取
- **数字疗法 (DTx)**：
    - 定义：经临床验证的软件驱动干预方案（Software as a Drug） [支持:: [[Concept_数字疗法_DTx]]]。
    - 案例：[[Entity_Pear-001]] (失眠数字疗法)。
- **真实世界证据 (RWE)**：
    - 价值：作为 RCT 的补充，驱动加速审批与适应症扩展 [支持:: [[Concept_RWE]]]。
    - 技术支撑：需要 **[[Entity_OMOP_CDM]]** 等标准化模型支撑。
- **角色变化**：出现了 [[Entity_个案管理师]] 等负责院外管理的专业角色。

## 洞察与评估
- **围墙坍塌**：医疗服务从“离散的院内时点”向“连续的全生命周期”跨越。
- **证据链重构**：数字化不仅提供了效率，更提供了关于“疗效”的全新证据生成路径。
