# Concept_数字疗法_DTx

## 定义
数字疗法（Digital Therapeutics, DTx）是指由软件程序驱动的、基于循证医学证据的干预方案，用以预防、管理或治疗疾病。它被形象地称为“软件即药物（Software as a Drug）”[^1]。

## 核心特征
1.  **循证性**：必须经过严格的临床试验（RCT）验证其有效性与安全性。
2.  **合规性**：需获得医疗器械监管部门（如 NMPA, FDA）的二类或三类注册证。
3.  **处方权**：通常由医生开具处方，作为传统药物或疗法的补充或替代。
4.  **行为干预**：核心机制往往基于认知行为疗法（CBT）、生活方式干预或生物反馈。

## 典型案例
- **[[Entity_Pear_Therapeutics.md]]**：其产品 reSET (药物滥用) 和 Somryst (慢性失眠) 是全球 DTx 的标杆[^1]。

## 行业趋势
- **支付方变革**：从患者自费转向商业保险与雇主支付。
- **全周期管理**：DTx 正在拆掉医院围墙，实现从院内诊疗到院外全周期的连续干预 [Supports:: [[Concept_Medical_Fortress_Four_Walls.md]]]。
