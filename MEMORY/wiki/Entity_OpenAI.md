# OpenAI

## 概览
OpenAI 是一家处于全球领先地位的人工智能研究实验室。其开发的 GPT 系列模型是医疗数字化领域的核心评估基准。

## 医疗领域表现与风险
在 2026 年的一系列临床审计中，OpenAI 的通用模型表现出了明显的二元性：
1. **高性能基准**: 在处理结构化电子病历数据时表现卓越，准确率可达 94.9%[^1]。
2. **致死性漏诊风险**: 在急诊分诊场景下，面对开放式人类交互，漏诊率高达 **50%-52%**，表现出明显的安全性漏洞 [反驳:: [[Concept_Probability_Machine.md]]][^1]。

## 战略演进
OpenAI 的 Frontier 架构正在被医疗机构作为基础底座，但其“黑盒”特性促使医疗机构加速建设 **[[Concept_MSL_医疗语义层.md]]** 以实现安全拦截。
