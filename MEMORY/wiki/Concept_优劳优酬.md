# 优劳优酬 (Value-based Pay)

## 核心内涵
**优劳优酬** 是中国公立医院绩效分配制度改革的核心方向。它主张彻底脱离以“收入提成”为导向的旧模式，转而建立一套基于“工作量、难度、风险、质量与资源消耗”的多维评价体系，旨在激励医务人员追求技术价值而非单纯的检查收入。

## 技术工具
- **RBRVS (以资源消耗为基础的相对价值比率)**: 衡量技术劳务价值的度量衡。
- **DRG 绩效评价**: 引入 CMI (病例组合指数) 衡量诊疗难度。
- **SMART 指标**: 结合关键质量控制点 (KPI) [支持:: [[Concept_SMART_价值指标.md]]]。

## 与数字化的关系
优劳优酬的落地极度依赖于 **[[Concept_业财融合.md]]** 与 **[[Entity_HRP.md]]** 的支撑。只有通过系统自动采集全量业务数据，才能确保考核的客观性、实时性与可回溯性 [支持:: [[Source_第十二讲：医院的“大管家”：新一代医院资源规划（HRP）系统.md]]]。

## 战略意义
它是推动医院从“规模扩张”转向“内涵式增长”的指挥棒。通过重塑激励罗盘，让“好医生”获得应有的体面收入，从而驱动临床路径的持续优化与运营效率的本质提升。
