# Source: 序言：在迷雾中绘制地图

## 核心摘要
作为《医疗数字化三十讲》的开篇，本文定义了数字化转型中的“迷雾”状态，并提出了“认知向导”的角色转型需求。在技术爆发与政策高压的双重挤压下，医疗行业急需一张清晰的逻辑地图[^1]。

## 关键观点
1.  **认知向导 (Cognitive Navigator)**: 数字化顾问应从推销功能转向管理客户的认知资源，降低理解成本 [支持:: [[Entity_医疗数字化顾问.md]]]。
2.  **隐藏脚手架**: 优秀的咨询应为客户呈现思想的纪念碑，而非展示分析过程的杂乱脚手架[^1]。
