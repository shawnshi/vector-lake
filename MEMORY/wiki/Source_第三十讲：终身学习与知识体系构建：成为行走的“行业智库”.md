# Source | 终身学习与知识体系构建

## 核心概述
本讲探讨了在医疗 IT 这一高熵值行业中，咨询顾问如何通过构建个人知识管理系统 (PKM) 抵御 **知识半衰期** 的压力[^1]。核心方法论是“输出驱动学习”。

## 关键提取
- **知识半衰期 (Knowledge Half-life)**：医疗 IT 知识迭代极快，传统经验迅速贬值[^1]。
- **学习闭环**：输入 (Input) -> 处理 (Processing) -> 连接 (Connection) -> 输出 (Output)[^1]。
- **工具链**：强调卡片笔记、双向链接与结构化写作在内化知识中的作用[^1]。
- **顾问定位**：被定义为“技术翻译官”与“行走的行业智库”[^1]。
