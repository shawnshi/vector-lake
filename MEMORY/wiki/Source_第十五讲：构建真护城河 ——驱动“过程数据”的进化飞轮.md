# 第十五讲：构建真护城河 ——驱动“过程数据”的进化飞轮

## 核心内容摘要
本讲重新定义了医疗 AI 的核心资产。作者提出，传统的“存量事实数据”（病历、报告）因基础模型的通用化而贬值，真正的护城河在于“**过程数据 (Flow Process Data)**”[^1]。

过程数据的定义：
- 它记录了 AI 给出建议、用户进行修正、以及最终采纳结果的全闭环轨迹[^1]。
- 它捕获了专家在标准病历之外的“隐性知识”和机构特定的“临床潜规则”[^1]。

## 关键提取
- **进化飞轮 (Evolution Flywheel)**：模型嵌入工作流 -> 捕获交互数据 -> 场景化微调 -> 效果增强 -> 粘性增强的增长闭环[^1] [支持:: [[Concept_Evolution_Flywheel.md]]]。
- **竞争壁垒**：过程数据具有高度的排他性，因为它是模型与特定机构、特定科室工作流“共同进化”的结果 [支持:: [[Concept_Flow_Process_Data.md]]]。
