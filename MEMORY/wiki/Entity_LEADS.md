# Entity: LEADS

## 概述
LEADS 是专为医疗领域设计的特定基础模型（Foundation Model），其核心功能聚焦于**高精度医学证据合成**（Medical Evidence Synthesis）。

## 技术特征
- **抗幻觉检索**：通过深度集成专业医学文献库，LEADS 在合成循证医学证据时表现出远超通用大模型的确定性[^1]。
- **结构化输出**：能够将零散的临床研究自动转化为符合 PRISMA 标准的综述草案。
