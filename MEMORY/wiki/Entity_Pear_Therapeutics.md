# Entity: Pear Therapeutics

## 描述
Pear Therapeutics 是全球数字疗法（DTx）领域的先驱性公司。其最知名的产品 reSET 和 reSET-O 获得了 FDA 的上市许可，用于治疗药物使用障碍[^1]。

## 标杆案例
- **Pear-001**: 用于慢性失眠症的数字疗法，是行业内研究数字化行为干预的标准范式。

## 行业地位
Pear 被视为 DTx 商业化的“风向标”，其发展历程（包括上市及后续的财务波动）为整个行业探索“谁来买单”提供了宝贵的先验经验 [关联:: [[Concept_数字疗法_DTx.md]]]。
