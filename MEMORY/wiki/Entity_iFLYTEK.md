# 科大讯飞 (iFLYTEK)

## 核心定位
科大讯飞是中国智能语音及大模型领域的龙头。在医疗领域，其核心优势在于强大的语音识别底座与广泛的基层医疗覆盖[^1]。

## 医疗产品线
- **智医助理**：深度绑定基层公卫与全科辅助诊疗。
- **语音电子病历**：在普适性科室占据核心市场份额。
- **ACI 演进**：通过星火医疗大模型，正将其语音插件升级为具备感知与生成能力的 [[Concept_ACI_环境感知临床智能.md]] 方案[^1]。

## 竞争策略
- **规模优势**：利用渠道优势执行“大单品”战略，占据基层市场。
- **入口主权**：通过掌握医生的语音输入入口，试图在传统 EMR 之上建立一层新的交互主权[^1]。
