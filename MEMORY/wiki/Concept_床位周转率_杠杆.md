# Concept: 床位周转率 (Bed Turnover Rate) 作为财务杠杆

## 定义
床位周转率是指在一定时期内平均每张病床的使用次数。在数字化语境下，它被重构为衡量医院智慧化程度与经济韧性的核心指标。

## 核心逻辑
1. **DRG/DIP 压力：** 在固定费率支付模式下，住院天数越多，医院边际收益越低甚至亏损。
2. **数字化解法：** 通过 [[Concept_Inpatient_Service_Experience.md]] 中提到的流程优化（入院服务中心、自助服务、床旁结算、智能调度），缩短非医疗性的术前/术后等待时间[^1]。
3. **住店化效应：** 将“住院即住店”的理念引入流程设计，通过消灭“信息黑箱”和行政性痛苦，实现患者快速流转与床位效能的极致压榨[^1]。
4. **杠杆效应：** 提升周转率意味着在相同的床位资源下，可以收治更多患者，从而增加总收入并对冲单次住院成本的上升[^1]。
