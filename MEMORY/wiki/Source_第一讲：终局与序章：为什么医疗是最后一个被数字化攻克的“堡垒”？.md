# Source: 第一讲：终局与序章：为什么医疗是最后一个被数字化攻克的“堡垒”？

## 核心摘要
探讨了医疗行业数字化的极高门槛。医疗由于其信息高度不对称、支付与使用分离、强监管以及价值非标的特性，成为了数字化最后攻克的“堡垒”[^1]。

## 关键观点
1.  **医疗堡垒四面墙**: 信任不对称、支付分离、强监管、价值非标 [支持:: [[Concept_Medical_Fortress_Four_Walls.md]]]。
2.  **数字化的终局**: 不是取代医生，而是通过增强信任与透明度，降低系统的总熵[^1]。
