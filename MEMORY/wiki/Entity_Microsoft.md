# Entity: Microsoft (微软医疗 AI)

## 定义
微软通过整合 **Nuance** 的临床记录技术与 **OpenAI** 的大模型能力，已成为医疗 AI 基座市场的核心玩家。

## 2026 战略产品线
- **DAX Copilot**: 环境感知临床智能 (ACI) 的代表作，实现临床对话的自动结构化记录[^1]。
- **Copilot Health**: 扮演“医疗导航员”角色，通过虹吸消费者数据（如 Teams, M365 中的健康信息）与临床数据对接。
- **算力租约**: 通过 Azure Health 平台，为医疗机构提供经过安全加固的 AI 推理环境。
