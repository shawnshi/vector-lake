# Concept: Value Proposition Matrix (价值主张矩阵)

## 定义
一种用于评估医疗 IT 产品商业生命力和市场定位的 MECE（相互独立、完全穷尽）分析工具。通过两个维度将产品划分为四个象限：
1. **服务对象**: 是服务于临床一线（降本）还是服务于管理者（增效/控费）。
2. **价值深度**: 是满足基础合规（刚需）还是创造额外溢价（数据价值）[^1]。

## 四象限模型
- **核心刚需**: 解决“如果不做就没法开门”的问题（如基础 HIS）。
- **生存杠杆**: 解决院长生存问题（如 **[[Concept_DRG_DIP.md]]** 质控）。
- **效率倍增器**: 真正解放医护生产力（如 **[[Entity_CIS.md]]** 中的 AI 辅助）。
- **未来溢价**: 发掘数据深层价值（如 **[[Concept_RWE.md]]**）。
