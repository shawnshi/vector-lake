# Concept: 智能体工厂 (Agent Factory)

## 定义
智能体工厂是一种集成化的开发与部署环境，旨在支持医疗机构规模化地构建、定制和运行自主 AI 智能体。它允许用户将医疗逻辑（如临床指南、费控规则）转化为可执行的 AI 资产。

## 核心特性
- **逻辑资产化**: 将零散的医疗知识打包为可调用的智能体，实现智力的标准化交付。
- **低代码定制**: 允许非技术背景的医生或管理人员根据本地需求调整智能体逻辑。
- **安全隔离**: 确保第三方或自定义 Agent 在受控环境下访问敏感的 EHR 数据[^2]。

## 行业标杆
- **[[Entity_Epic_Systems.md]]**: 推出了行业首个成熟的 Agent Factory 平台。其核心逻辑是：既然大模型是“概率机器”，那么 Epic 必须提供一个确定性的“外壳 (Harness)”和“工厂 (Factory)”，让医院在受控范围内开发智能体[^1][^2]。
