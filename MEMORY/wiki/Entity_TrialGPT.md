# TrialGPT

## 概述
由 **[[Entity_NIH]]** 开发的临床试验匹配系统，利用大语言模型自动解析患者病历并与复杂的临床试验入组/排除标准进行匹配。

## 核心指标
- **匹配精度**: 在自动筛选中达到 87.3% 的准确率 [^1]。
- **效率提升**: 显著缩短了患者从入组初筛到临床评估的时间周期。
