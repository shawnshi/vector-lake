# Source | 医疗数字化商业模式全景解析

## 核心概述
深度拆解医疗数字化的底层商业逻辑，明确了四大核心支付方的动力来源，并提出了“价值主张四象限”模型[^1]。

## 关键提取
- **核心支付方矩阵**：
    - **国家医保局 (NHSA)**：驱动政策性刚需与精细化管理[^1]。
    - **医院**：为评级与控费买单[^1]。
    - **药械企业**：渴求 RWE 以支撑准入与研发[^1]。
    - **患者/商保**：关注服务确定性与体验[^1]。
- **价值主张四象限**：运营优化 (省钱)、临床赋能 (效率)、市场准入 (赚钱)、患者服务 (体验)[^1]。
- **商业转型困境**：从一次性项目制向“效果付费”跃迁的归因难题与安全红线[^1]。
