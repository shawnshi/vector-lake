# 第四讲：蓝图的共识：从“信息化”到“数字化”再到“智慧化”的演化路径

## 核心摘要
本讲厘清了医疗 IT 发展过程中的三大阶段：信息化、数字化与智慧化。文章强调认知对齐是制定战略蓝图的前提，混淆这三个阶段会导致执行层面的动作变形。

## 关键提取
- **三大阶段定义**:
    - **信息化 (Informatization)**: 流程电子化，核心是“记录”，解决“有没有纸”的问题。
    - **数字化 (Digitization)**: 数据资产化与业务连接，核心是“流动”，解决“数据驱动决策”的问题。
    - **智慧化 (Intelligentization)**: AI 自主预测与学习，核心是“洞察”，解决“机器辅助决策”的问题 [支持:: [[Concept_信息化_数字化_智慧化演化路径.md]]][^1]。
- **演化不可逾越**: 没有稳固的信息化记录，数字化就成了“无源之水”；没有数字化的数据治理，智慧化就是“空中楼阁”[^1]。
- **地基决定上层建筑**: 医院应根据自身的底层数据能力（如 CDR/ODR 的成熟度）来规划演进阶梯。

## 与现有知识库的联系
