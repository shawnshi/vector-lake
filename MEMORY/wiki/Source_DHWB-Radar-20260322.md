# DHWB-Radar-20260322 (Source Summary)

本期雷达聚焦 Oracle Health (原 Cerner) 的战略收缩与数字化底座转型。

## 关键提取与发现

*   **Oracle Health 剧烈重构**: 
    *   **大幅裁员**: 爆出 30% 的大规模裁员，反映出其正在加速剥离低效的传统 HIS 业务。
    *   **资金流向**: 资源正在从传统业务抽离，转向 OCI (Oracle Cloud Infrastructure) 云底座与 AI 数据中心建设[^1]。
*   **算力房东模式 (Compute Landlord)**: Oracle 正在通过强制云化策略，将软件售卖模式转变为长期的算力与数据“房租”收取模式[^1]。
*   **市场动态**: 巨头正在利用 AI 作为优化资产负债表、剥离沉重传统包袱的工具。
