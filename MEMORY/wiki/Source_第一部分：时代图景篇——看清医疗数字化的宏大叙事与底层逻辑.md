# Source: 第一部分：时代图景篇——看清医疗数字化的宏大叙事与底层逻辑

## 核心摘要
对医疗数字化转型的宏观背景进行了系统性梳理，强调了从“零件堆砌”向“体系构建”转变的重要性[^1]。

## 关键观点
1.  **剥离幻觉**: 数字化不再是技术的炫耀，而是对物理真理的回归[^1]。
2.  **从功能到体系**: 医疗 IT 行业需要超越单一软件的交付，转向对整体业务逻辑的数字化重构。
