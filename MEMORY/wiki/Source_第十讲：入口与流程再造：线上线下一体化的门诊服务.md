# 第十讲：入口与流程再造：线上线下一体化的门诊服务

## 核心摘要
本讲探讨了智慧服务标准的落地实践，重点在于门诊流程的数字化再造。核心理念是“消灭物理流程”，通过数字入口矩阵（微信、App、自助机）实现患者与医院的“持续连接”。

## 关键提取
- **数字入口矩阵**: 核心价值不在于引流，而在于通过实名化身份认证实现与患者的闭环交互[^1]。
- **线上线下一体化**: 打破物理围墙，将预约、支付、检查预约、结果查询等功能全部数字化，减少患者在院内的无效跑动 [支持:: [[Concept_Inpatient_Service_Experience.md]]][^1]。
- **服务即连接**: 智慧服务的终局是建立全生命周期的健康管理关系，而非单次的挂号交易[^1]。
- **流程反思**: 如果一个功能仅仅是把纸质表单搬到屏幕上，而不改变背后的业务逻辑，那就不是数字化，只是电子化。
