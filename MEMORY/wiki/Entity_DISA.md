# Entity: DISA (美国国防信息系统局)

## 描述
DISA 是 [[Entity_DoD.md]] 下属的战斗支援机构，负责管理国防部范围内的信息系统。

## 数据治理贡献
- **全生命周期管理**: DISA 发布的指南将数据管理分为 8 个核心阶段，为医疗领域构建 [Concept_Data_as_a_Product] 提供了工程参考模型[^1]。
- **安全防务**: 其安全合规标准是全球高安全要求系统的重要参考。
