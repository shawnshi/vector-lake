# Concept: 医疗堡垒四面墙 (Medical Fortress Four Walls)

## 定义
描述医疗数字化转型极度艰难的物理基础框架。这四面墙定义了变革者必须面对的“阻力场”[^1]：

1.  **信任不对称 (Trust Asymmetry)**: 医患之间、管理者与临床之间极高的专业壁垒。
2.  **支付分离 (Payment Separation)**: 使用者（医生/患者）与支付者（医保/财政）不是同一主体，导致动力脱节。
3.  **强监管 (Heavy Regulation)**: 涉及生命安全，任何细微改动都面临严苛的法律与准入审查。
4.  **价值非标 (Non-standard Value)**: 医疗成果难以像工业产品一样被单一指标完美度量。

## 战略启示
数字化产品必须能够穿越甚至“利用”这些高墙，而非试图简单地推倒它们。例如，通过 **[[Concept_Evidence_Mesh.md]]** 来增强信任[^1]。
