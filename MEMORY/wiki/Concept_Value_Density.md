# Concept: 数据价值密度 (Value Density)

## 定义
衡量单位数据量所能提供的临床决策支持或管理效益的指标。在 LLM 时代，盲目追求数据量（Volume）往往会导致信噪比下降，真正的竞争力在于挖掘和利用“高价值密度”的私有数据[^1]。

## 核心内涵
- **信噪比**: 剔除冗余、错误与无用信息后的纯净度。
- **关联度**: 数据与患者结局、运营成本、质控指标的因果关联强度。
- **结构化程度**: 机器可读性越强，价值密度越高 [支持:: [[Concept_数据结构化.md]]]。
