# Source: 模块一：第一性原理 —— 洞见权力、风险与成本的本质

## 核心摘要
本模块解构了医疗 AI 的物理与社会学本质。师成指出，LLM 是典型的“概率机器”，其内生的幻觉与不确定性在医疗领域构成了致命的“责任黑洞”。解决路径不在于单纯提升算法，而在于通过“架构包围算法”实现责任确权与风险对冲[^1]。

## 关键观点
- **概率机器 (Probability Machine)**: LLM 的本质是 token 预测，幻觉是其功能而非 bug [支持:: [[Concept_Probability_Machine]]]。
- **责任黑洞 (Responsibility Black Hole)**: 当 AI 出错时，决策链的模糊性导致了责任消散。必须通过 [Concept_Evidence_Mesh] 锁定证据网来填补此黑洞 [支持:: [[Concept_Responsibility_Black_Hole]]]。
- **初稿成本革命**: AI 带来的核心红利是“初稿成本”的极速降低，而非“最终决策”的自动化。
- **架构包围算法 (Architecture Surrounds Algorithm)**: 通过 RAG、MSL 和强约束外壳，将概率性的算法锁定在确定性的物理围栏中 [支持:: [[Concept_架构包围算法]]]。
- **权力分配博弈**: AI 会扰动医院原有的利益分配。一个成功的方案必须回答：谁被赋能了？谁的权力被削弱了？谁在承担最终风险？
