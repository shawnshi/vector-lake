# 高质量医疗数据集建设：现状、挑战与未来路径

## 核心摘要
本文系统性地界定了“高质量医疗数据集”的战略地位与构建路径。高质量数据不再是业务的副产品，而是 AI 的生命线。其核心范式已从“数据囤积”转向“数据即产品 (DaaP)”与“战略狩猎”模式[^1]。

## 关键观点
- **三高特征 (3-Highs)**:
  - **高价值应用**: 围绕临床高频痛点进行“战略狩猎”，而非被动收集。
  - **高知识密度**: 包含深度专家标注、临床路径映射与知识图谱对齐。
  - **高技术含量**: 涉及 [Entity_OMOP_CDM] 映射、标准化清洗与隐私计算对齐。
- **数据即产品 (DaaP)**: 强调数据的可发现性、互操作性与生命周期管理 [支持:: [[Concept_Data_as_a_Product]]]。
- **VAULTIS 准则**: 引入 [[Entity_DoD]] 的数据战略标准（可见、可及、可理解、可关联、可信、可互操作、安全）作为衡量高质量的物理指标 [支持:: [[Concept_VAULTIS]]]。
- **战略猎杀 (Strategic Hunting)**: 数据集的建设应遵循“场景驱动”而非“技术驱动”，优先攻克那些能产生“数据飞轮”效应的核心领域。
