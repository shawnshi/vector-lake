# Entity: 欧洲医疗数字化模式

## 概况
欧洲模式以单一支付方（Single-Payer）制度下的行政强规划为核心，强调医疗的社会福利属性与公民数据主权。

## 核心特征
1. **行政强规划**：以英国 NHS 为代表，通过顶层设计强制推行统一的数据标准与服务架构[^1]。
2. **数据主权**：极度重视公民隐私与数据所有权。
3. **创新困境**：由于过度依赖中央规划，局部市场的竞争与创新活力往往弱于美国模式。
