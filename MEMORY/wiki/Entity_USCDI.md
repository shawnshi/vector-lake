# Entity: USCDI

## 描述
USCDI (United States Core Data for Interoperability) 是由美国 ONC 制定的一套标准化的健康数据类和元素，旨在支持跨系统的患者数据交换。

## 战略价值
- **高质量数据集基石**: 为医疗 AI 的模型训练与验证提供了具备互操作性的“黄金基准”[^1]。
- **语义层对齐**: 它是 [Concept_MSL_医疗语义层] 实现全球化语义对齐的重要参考标准。
- **VAULTIS 支撑**: 直接支持了 [Concept_VAULTIS] 框架中的“可互操作性”与“可理解性”要求[^1]。
