# Source: 第八讲：医生的“眼睛”和“画笔”：PACS 与 LIS 系统详解

## 核心内容摘要
本讲深度解析了 PACS 和 LIS 两大核心医技系统，强调其从“辅助工具”向“决策引擎”的演进。

### 关键洞察
1. **PACS（医生的眼睛）**：正从“胶片仓库”转变为“影像数据银行”，其核心价值在于影像的高速调阅与 AI 辅助诊断 [衍生于:: [[Entity_PACS.md]]]。
2. **LIS（医生的画笔）**：从“化验单工厂”转变为“健康罗盘”，强调检验结果的实时交互与闭环管理 [衍生于:: [[Entity_LIS.md]]]。
3. **决策化趋势**：医技系统必须无缝嵌入临床工作流，提供即时数据支持[^1]。
