# Entity: OHDSI

## 描述
OHDSI (Observational Health Data Sciences and Informatics, 读作 Odyssey) 是一个多学科、跨国界的开源协作社区，旨在通过大规模数据分析来改善医疗健康[^1]。

## 核心贡献
- **OMOP CDM**: 维护并推广通用数据模型，使其成为全球真实世界研究的“世界语” [支持:: [[Entity_OMOP_CDM.md]]]。
- **开源工具栈**: 开发了如 ATLAS 等工具，帮助研究者在不暴露隐私数据的情况下进行多中心分析。

## 使命
通过建立全球性的观测数据网络，支撑高质量 [[Concept_RWE.md]] 的产生，实现医学研究的民主化与规模化。
