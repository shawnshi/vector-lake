# Source: 第四讲：数据资产负债表 —— LLM的“燃料”与“负债”

## 核心摘要
本讲提出了“数据资产负债表”框架，旨在引导医院管理者建立冷静的数据观。在 LLM 时代，数据不仅是驱动智能的“燃料”，其安全性、合规性与低质量也构成了沉重的“负债”[^1]。

## 关键观点
1.  **数据的二元性**: 高质量、结构化的临床数据是核心资产；未经清洗、存在合规隐患的数据则是潜在负债[^1]。
2.  **价值密度**: 数据的价值不在于体量（Volume），而在于其与临床结果、经济效益的关联密度 [支持:: [[Concept_Value_Density.md]]]。
3.  **可用而非所有**: 提倡通过技术手段释放数据价值，而非盲目追求物理占有 [支持:: [[Concept_Usable_not_Ownable.md]]]。
