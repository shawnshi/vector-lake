# Entity: 美国医疗数字化模式

## 概况
美国医疗数字化模式是以市场竞争与资本效率为核心驱动力的典型代表。

## 核心特征
1. **寡头垄断**：以 [[Entity_Epic_Systems.md]] 和 Oracle Health (Cerner) 为双核。
2. **商业机器属性**：系统设计初衷是为保险计费与资本回报最大化服务的“商业运营机器”[^1]。
3. **厂商锁定 (Vendor Lock-in)**：由于缺乏顶层行政干预，形成了事实上的数据烟囱与高昂的迁移成本。
4. **HIE 困境**：尽管技术先进，但跨机构的互联互通（Health Information Exchange）在商业博弈下推进艰难。
