# Source: 第四部分：顾问素养篇——成为卓越的医疗数字化咨询专家

## 概述
本文系统阐述了医疗数字化顾问在行业变革期应当具备的核心素养与方法论。提出顾问不再是单纯的“软件销售”，而是通过“医生-建筑师-教练”的复合角色，实现医疗 IT 项目从愿景到落地的价值闭环。

## 核心观点
1. **顾问三重角色**：
    - **医生 (Diagnostician)**：具备穿透症状、诊断行业病根的逻辑洞察力。
    - **建筑师 (Architect)**：具备设计四大架构（业务、应用、数据、技术）的工程能力。
    - **教练 (Coach)**：具备伴随式项目管理与认知对齐的沟通能力。
2. **结构化思维**：利用逻辑树与 MECE 原则拆解医疗系统的复杂性。
3. **价值闭环**：数字化项目的成功取决于能否将 What（目标）转化为 How（路径）并最终交付 ROI。
