# TrialGPT

## 概览
TrialGPT 是由 **[[Entity_NIH.md]]** 开发的专门用于临床试验患者匹配的人工智能 Agent。

## 核心功能
1. **入组筛选**: 自动化解析患者病历并与复杂的临床试验准入标准进行匹配。
2. **高精度**: 实测匹配精度达到 **87.3%**[^1]。

## 战略价值
TrialGPT 的成功案例证明了 AI 在科研垂直领域的巨大潜力，是 **[[Concept_Evidence_Mesh.md]]** 在科研领域的成功实践。
