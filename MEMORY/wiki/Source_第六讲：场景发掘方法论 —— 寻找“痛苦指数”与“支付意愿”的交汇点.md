# 第六讲：场景发掘方法论 —— 寻找“痛苦指数”与“支付意愿”的交汇点

## 核心观点
本讲提供了一套寻找 AI 爆发场景的量化评估方法论，核心逻辑在于寻找组织内部痛点与外部政策压力的“共振点”[^1]。

## 关键内容
- **痛苦指数 (Pain Index) 2.0**：不仅评估场景的频率、时长和枯燥度，更强调识别那些让医院感到“肉疼”的运营出血点[^1]。
- **网络价值乘数 (Network Value Multiplier)**：引入了生态视角，衡量场景解决后对外部利益相关者（医保局、卫健委）的价值贡献。乘数越高，支付意愿越强[^1]。
- **寻找杠杆点**：理想场景需具备“高度模板化、高频耗时、低风险、连接孤岛”四大特征。如 DRG 控费场景即为痛苦指数与乘数的双高点[^1]。
